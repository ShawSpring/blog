## ref操作DOM

使用`ref`引用DOM节点不必多说，这里说明了`ref`的两个特性： 
1. ref可以跨渲染保留
2. `ref.current`更改不会触发重新渲染


 DOM 操作是 ref 最常见的用例，但 useRef Hook 可用于存储 React 之外的其他内容，例如计时器 ID 。
 与 state 类似，ref 能在渲染之间保留。你甚至可以**将 ref 视为设置它们时不会触发重新渲染的 state 变量**！
